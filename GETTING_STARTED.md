## 🚀 Three-Step Quick Start

### ⏱️ Takes 3 minutes!

**Step 1: Install Node.js**
```
1. Go to https://nodejs.org/
2. Click "Download LTS"
3. Run installer
4. Restart your computer
```

**Step 2: Install Dependencies**
```bash
npm install
```

**Step 3: Start Server**
```bash
npm start
```

Done! Visit: http://localhost:3000

## 🆘 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| "Where do I start?" | Read START_HERE.md |
| "Node.js not found?" | Install from nodejs.org, restart |
| "npm install fails?" | Delete node_modules, try again |
| "Port already in use?" | Change PORT in .env |
| "Database connection error?" | Check .env credentials |
| "Login not working?" | Check browser console (F12) |
